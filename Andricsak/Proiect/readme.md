# Rent a game project 
#### Domeniul:
- Vanzarilor
- Aplicatie web cu lista jocurilor disponibile pentru inchiriere.

##### Functionalitati:
- Inregistrarea unui nou user
- Adaugarea jocurilor in baza de date de catre admin
- Inchirerea unui joc
- Returnarea unui joc si extinderea perioadei de inchiriere
